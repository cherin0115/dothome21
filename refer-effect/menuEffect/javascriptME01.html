align-items 속성

align-items 속성은 콘텐츠 아이템의 내부 상하관계 정렬 상태를 설정합니다.

align-self 속성

align-self 속성은 개별적인 콘텐츠 아이템의 정렬 상태를 설정합니다.

all 속성

all 속성은 요소의 속성을 초기화 또는 상속을 설정합니다.

animation 속성

animation 속성은 애니메이션과 관련된 속성을 일괄적으로 설정합니다

animation-delay 속성

animation-delay 속성은 애니메이션 지연 시간을 설정합니다.

    const tabBtn1 = document.querySelectorAll(".view-title ul li");
    const tabCont1 = document.querySelectorAll(".view-cont > div");

    tabBtn1.forEach((element, index) => {
        element.addEventListener("click", function () {
            tabBtn1.forEach(el => {
                el.classList.remove("active");
            })
            element.classList.add("active");

            tabCont1.forEach(el => {
                el.style.display = "none";
            })

            tabCont1[index].style.display = "block";
        })
    })
